# Seoullure – Shopify E-Commerce Store

A premium Shopify e-commerce store built for **Seoullure**, a Korean skincare brand selling authentic K-beauty products in South Africa.

🌐 Live Website: https://seoullure.co.za

---

## Project Overview

Seoullure is an e-commerce store designed to provide a clean, elegant and mobile-friendly shopping experience for Korean skincare products.

The project was built on Shopify to allow for fast deployment, secure checkout, easy product management and low-maintenance operation for the client.
