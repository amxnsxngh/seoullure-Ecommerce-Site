# Project Overview

## Client

Seoullure

## Industry

E-commerce / Korean Beauty / Skincare

## Platform

Shopify

## Website

https://seoullure.co.za

## Goal

To create a professional online store for selling Korean skincare products in South Africa.

## Why Shopify Was Used

Shopify was selected because it allows for:

- Faster launch timeline
- Secure hosting
- Easy product management
- Built-in checkout
- Simple order management
- Lower maintenance compared to a fully custom website
- Scalability as the business grows

## Key Deliverables

- Shopify store setup
- Theme customisation
- Product import preparation
- Homepage design
- SEO configuration
- Brand implementation
- Navigation setup
- Store readiness for launch
